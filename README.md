# twterm

A full-featured CLI Twitter client

## Screenshot

![screenshot](http://twterm.ryota-ka.me/screenshot.png)

## Requirements

- Ruby (compiled with C Curses and Readline)
- C curses
- Readline

## Installation

```
$ gem install twterm
```

## Execution

```
$ twterm
```
